For Matlab version 9.4 (R2018a) or higher

English part:
This program is for generating cards from a set of pieces.
On the github, the pieces are in the parts_of_map.rar archive, you will need to unpack it;)
Most files are functions that were used for development and testing.
In the end, for the correct operation, enough functions:
fmap_all.m
showmemap.m
as well as archive pieces
parts_of_map.rar
and three sets:
Riv5_23.mat
Riv5_27.mat
Riv5_w.mat


Русская часть:
Эта программа для генерации карты из набора кусочков.
На гитхабе кусочки лежат архивом parts_of_map.rar, нужно будет распаковать ;)
Большинство файлов - функции, которые использовались для разработки и тестирования.
В конечном итоге для корректной работы достаточно функций:
fmap_all.m
showmemap.m
а так же архива кусочков 
parts_of_map.rar 
и трех наборов:
Riv5_23.mat
Riv5_27.mat
Riv5_w.mat
